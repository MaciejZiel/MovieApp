# MovieApp

A simple Python application for managing and working with movie data stored in JSON files.

This project demonstrates basic object-oriented programming in Python combined with JSON data handling.
It is intentionally minimal and easy to extend.

---

## Project structure

- `main.py` — application entry point
- `movie.py` — `Movie` class definition
- `movie_manager.py` — logic for loading and managing movies
- `movies.json` — sample movie dataset
- `movies2.json` — additional sample dataset

---

## Features

- Load movie data from JSON files
- Represent movies as Python objects
- Clean and readable project structure
- Easy to extend with new functionality

---

## Requirements

- Python **3.8+**
- No external dependencies

---

## How to run

1. Check Python version:

```bash
python --version
```

2. Run the application:

```bash
python main.py
```
